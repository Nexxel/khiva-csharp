static array.Array khiva.normalization.Normalization.DecimalScalingNorm ( array.Array  tss)
inlinestatic

Normalizes the given time series according to its maximum value and adjusts each value within the range (-1, 1).

Parameters
tssExpects an input array whose dimension zero is the length of the time series (all the same) and dimension one indicates the number of time series.
Returns
An array with the same dimensions as tss, whose values (time series in dimension 0) have been normalized by dividing each number by 10^j, where j is the number of integer digits of the max number in the time series.

Normalizes the given time series according to its minimum and maximum value and adjusts each value within the range[low, high].

Parameters
tssExpects an input array whose dimension zero is the length of the time series (all the same) and dimension one indicates the number of time series.
highMaximum final value (Defaults to 1.0).
lowMinimum final value (Defaults to 0.0).
epsilonSafeguard for constant (or near constant) time series as the operation implies a unit scale operation between min and max values in the tss.
Returns
Array with the same dimensions as tss, whose values (time series in dimension 0) have been normalized by maximum and minimum values, and scaled as per high and low parameters.

static array.Array khiva.normalization.Normalization.MeanNorm ( array.Array  tss)
inlinestatic

Normalizes the given time series according to its maximum-minimum value and its mean. It follows the following formulae:

\[ \acute{x} = \frac{x - mean(x)}{max(x) - min(x)}. \]

Parameters
tssExpects an input array whose dimension zero is the length of the time series (all the same) and dimension one indicates the number of time series.
Returns
An array with the same dimensions as tss, whose values (time series in dimension 0) have been normalized by substracting the mean from each number and dividing each number by \( max(x) - min(x)\), in the time series.
