Calculates the k-means algorithm.

[1] S. Lloyd. 1982. Least squares quantization in PCM. IEEE Transactions on Information Theory, 28, 2, Pages 129-137. /summary>

Parameters
tssExpects an input array whose dimension zero is the length of the time series (all the same) and dimension one indicates the number of time series.
kThe number of means to be computed.
centroidsThe resulting means or centroids.
labelsThe resulting labels of each time series which is the closest centroid.
toleranceThe error tolerance to stop the computation of the centroids.
max_iterationsThe maximum number of iterations allowed.
